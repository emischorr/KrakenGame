
#### Ideen ####
git init

git config --local user.name (oder --global mit globaler highscore)

git clone level

git reset --soft (neustart level)
git reset --hard (neustart spiel)
git commit (level beenden/highscore speichern)

git stash / unstash (pause game)

git --help (Hilfe/Steuerung)

git garbage collection als Feinde zum Ausweichen

#### GAMEPLAY ####

- Spieler bewegt sich von unten nach oben
- oben angekommen ist das level beendet
- Punkte pro gerettetem Commit
- Garbage Collection als großer Pacman (Gegner) der sich nur vertikal bewegt (wenn kollision -> game over)
- Rebase als hindernis auf einem branch
